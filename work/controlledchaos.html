Controlled Chaos

Creative Coding, Generative Design, Interaction Design, Systems Thinking, Computational Design

An interactive generative system where user input shapes a continuously evolving particle field. Through presets, controls, and gestures, the project explores how structured rules can produce unpredictable, fluid visual behaviors in real time.

[JavaScript, Three.js, HTML / CSS, WebGL (via Three.js)]
